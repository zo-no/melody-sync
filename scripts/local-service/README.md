# Local Service Scripts

This directory owns local service lifecycle scripts for MelodySync.

- `common.sh`: shared OS/runtime helpers and proxy environment wiring.
- `setup.sh`: interactive local service installation.
- `start.sh`: start the local chat service.
- `stop.sh`: stop the local chat service.
- `restart.sh`: restart the local chat service.
- `logs.sh`: service status and log inspection.

Root shell files remain thin wrappers only.
